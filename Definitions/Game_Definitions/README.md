# Game_Definitions

该目录包含通过 **RTTI** 解析得到的《Cyberpunk 2077》所有定义信息。你可以在此检索 **枚举（Enums）**、**位域（Bitfields）**、**类（Classes）**、**结构体（Structs）** 以及 **全局函数（Global Functions）**。

对于每个类与结构体，都提供其 **属性列表** 与 **函数列表**；同时也支持查看 **父类/子类继承关系**，便于理解类型层级与扩展点。

## 用途建议

- 快速查找某个类型/函数的定义与签名
- 了解类/结构体的字段与方法组成
- 梳理继承链、定位扩展或覆写入口
