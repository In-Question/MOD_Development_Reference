# print()

Prints one or more objects to the [CET Console](https://wiki.redmodding.org/cyber-engine-tweaks/console/console) and [CET Console Logs](https://wiki.redmodding.org/cyber-engine-tweaks/first-steps/logs-debug#cet-console-logs) file. Multiple objects will be printed separated by a space.

## Definition

```swift
print(obj: object, [obj2: object, [...]]) -> nil
```

## Usage example

```lua
print('Hey', 'there', 1, 2, true)
```

Console output:
> Hey there 1 2 true
